React Renderers

A directory of the growing community of React Renderers, sorted alphabetically. Generally the links are either to the general GitHub repo, or the directory or file that includes the entry-point to a renderer implementation.

  • ReactAframe. AFrame is a WebVR library.
  • ReactART. ART is a retained mode vector drawing library.
  • ReactBlessed. Blessed is a high-level API for for building interactive terminal interfaces.
  • ReactDOM. The original React.js
  • ReactDOM/server. The original React.js
  • React Streaming DOM
  • ReactGibbon (ReactConf talk with reference). Gibbon is a proprietary rendering engine by Netflix to power many of their experiences.
  • ReactHardware. React bindings to the firmata protocol for interacting with hardware.
  • ReactNative. The original custom renderer. Build native iOS and Android apps with React.
  • ReactNativeDesktop. A fork of React Native for building OSX applications.
  • ReactShallowRenderer. A small renderer primarily used by ReactTestUtils.
  • React Symphony.
  • ReactThreeRenderer. Render rich 3D experiences with three.js
  • ReactTitanium. The Titanium SDK is a JS library to build cross-device experiences.
  • React Worker DOM. A high-quality experiment to improve React performance by running in a WebWorker.
  • React X11. A renderer targeting x11.
  • React Konsul. Browser console renderer.
  • mylittledom. High-level DOM-like terminal interface library.
  • react-pdf. Create PDF files using React
  • react-ionize. A React Fiber renderer for Electron apps
  • canvas-immediate Canvas Immediate mode rendering using React
  • hig-react-fiber React Renderer for hig-web — a part of Autodesk’s Orion UI framework.
